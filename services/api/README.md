## MultiApps API Service

This Express + Prisma service powers the MVP experiences for SecureGuard, Nova, Bloom, LedgerLoop, and StudyForge. It ships with authentication, domain endpoints, Prisma models, and Swagger documentation.

### Quick Start

```sh
cd services/api
cp env.example .env
npm install
npm run prisma:generate
npm run prisma:migrate -- --name init
npm run db:seed
npm run dev
```

Visit `http://localhost:4000/docs` for the interactive API reference.

### Scripts

| Script | Description |
| --- | --- |
| `npm run dev` | Start API with hot-reload |
| `npm run build` | Type-check and emit JS to `dist/` |
| `npm run test` | Run Vitest unit/integration tests |
| `npm run prisma:migrate` | Create or update local SQLite schema |
| `npm run prisma:deploy` | Apply migrations in production |
| `npm run db:seed` | Seed database with demo data |

### Domain Overview

- **SecureGuard**: `/security` incidents and remediation tasks
- **Nova**: `/nova` workflows, steps, guardrail simulations, and voice capsules
- **Bloom**: `/wellness` energy snapshots and journaling
- **LedgerLoop**: `/finance` accounts, transactions, autopilot rules, and forecasting
- **StudyForge**: `/study` sprints, focus sessions, tutor snapshots

All protected routes require a Bearer JWT generated by `POST /auth/login` or `POST /auth/register`.
